You are Replit Agent. Build a complete, production-ready full-stack web application called **ScholarForge** – a neutral research collaboration platform where scholars can create, manage, and collaborate on research projects.

### TECH STACK (use exactly this)
- Next.js 15 (App Router) with TypeScript
- Tailwind CSS + shadcn/ui + Framer Motion (for lively animations)
- Prisma ORM + PostgreSQL (Replit built-in database)
- Authentication: NextAuth.js v5 (Email/Password + Google OAuth)
- Real-time: Socket.io (full real-time chat with image upload)
- State management: Zustand
- File uploads: Replit App Storage (or Cloudinary if easier)
- Dark/Light mode with system preference

### CORE CONCEPT
- Completely neutral platform – no personal branding.
- Any user can sign up/sign in with email or Google.
- Creator of a project automatically becomes LEAD.
- 30-minute inactivity session timeout (auto logout if user is idle).
