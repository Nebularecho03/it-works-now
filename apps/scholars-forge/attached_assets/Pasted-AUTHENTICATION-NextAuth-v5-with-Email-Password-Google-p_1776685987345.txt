AUTHENTICATION

NextAuth v5 with Email/Password + Google provider
First user who signs up becomes ADMIN automatically
Session strategy: JWT
30-minute inactivity timeout: Implement client-side idle detection (mouse/keyboard movement). After 30 minutes of inactivity → automatic signOut and redirect to login.

NEW FEATURES ADDED (to make it feel alive & complete)

Chat Session Picture Upload
In every project chat, users can upload images (JPG, PNG, GIF) directly in the chat box.
Images are displayed inline with text messages.
Stored securely and shown with lazy loading.

Project Cover Images / Icons
When creating/editing a project, user can upload a cover image (thumbnail/icon).
Beautifully displayed on project cards and inside project dashboard.

Real-time "Alive" Features
Live online status (green dot + last active time) for team members in chat.
Typing indicators ("User is typing...").
Message reactions (❤️ 👍 👏 🎉 etc.) – clickable on any message.
Smooth Framer Motion animations on page transitions, modals, project cards, chat bubbles.
Live activity feed on dashboard (real-time updates when someone joins a project, uploads file, etc.).
Confetti animation when a project is marked "Completed".

Extra Suggested & Applied Features
In-app notification bell (shows new invitations, messages, project updates).
Searchable Scholar Directory (list of all users with filters).
Project export button (download project as JSON + all files in one zip).
Beautiful responsive sidebar (collapsible on mobile) with live project count badges.
Toast notifications for every action.
